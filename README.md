# Open Mover 
Game / Game framework in C++ using OpenGL (GLAD GLFW CGLM)

## Quick start
```console
  $ make main
  $ ./main
```

## Style
C++ files:
 - Method names: MoveRect();
 - Variable / argument names: move_rect(int xDiffrence)
